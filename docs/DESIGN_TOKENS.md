# Design Tokens (Mobile-first)

These tokens ensure readable, low-stress UI on small screens with strong contrast.

## Colors
- `--color-bg`: app background
- `--color-surface`: main cards / panels
- `--color-surface-2`: secondary cards / chips
- `--color-text`: primary text
- `--color-text-muted`: secondary text
- `--color-border`: default borders
- `--color-primary`: primary action
- `--color-danger`: overdue/alert
- `--color-warning`: soon/attention

## Typography
- Title: 14–16px, semibold
- Meta: 12px, regular
- Labels/chips: 10–11px, uppercase

## Spacing
- `--space-2`: 8px
- `--space-3`: 12px
- `--space-4`: 16px
- `--space-6`: 24px

## Radius
- `--radius-lg`: 16px (cards)
- `--radius-xl`: 20px (hero/primary)

## Shadows
- `--shadow-card`: subtle card shadow for separation
- `--shadow-float`: used for top “Next reminder”

## Button sizes
- Minimum tap target: 44–48px height
- Icon buttons: 40–44px

## Rationale
- Higher contrast for mobile readability.
- Reduced blur/gradient in navigation.
- Small, consistent chip sizes for scanning.
